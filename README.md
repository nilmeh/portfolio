## ğŸ¨ Customization

### Content Management
- Add projects in `content/projects/` as MDX files
- Add research papers in `content/research/`
- Add awards in `content/awards/`

### Styling
- Customize colors in `src/app/globals.css`
- Update component styling in shadcn/ui components
- Modify layout in respective page files

### Contact Information
Update contact details in:
- `src/app/page.tsx` (hero section)
- `src/app/about/page.tsx` (about page)
- `src/app/contact/page.tsx` (contact page)
