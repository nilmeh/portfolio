# Niloy Meharchandani - Portfolio

A modern, professional portfolio website showcasing my projects, research, education, and achievements as a Computer Science student at UCLA.


## Tech Stack

- **Framework**: Next.js 14 (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **UI Components**: shadcn/ui (Radix primitives)
- **Icons**: Lucide React
- **Content**: MDX for project documentation
- **Deployment**: Ready for Vercel deployment
